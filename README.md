# ECS

## Build

Build the ECS (will give you a static library) :
```
> ./MakeLibECS.sh
```
Build the example (you must have built the ECS before and have the SFML installed) :
```
> cd example
> make
```
Clean everything :
```
> ./Clean.sh
```
